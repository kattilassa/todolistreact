# TODO list with React and Firebase

## Description
A simple TODO application built with React and Firebase Realtime Database for Metropolia React.js Fundamentals-course.

## Live Demo
https://github.com/kattilassa/todolistreact

## Features
- Add TODO items
- Delete TODO items
- TODO items are stored in Realtime Database
- Data is fetched from Firebase API
- Data is displayed with AG Grid
- TODO items can be sorted and filtered.

## Tools
- React
- Vite
- Firebase Realtime Database
- GitHub Pages
